Ramyeon-sinusoidal function

A Sinusoidal function

Transformation:
I chose this photo because I often eat ramyeon as lunch near the Sunway’s convenience store. To model the sign appearing in this photo, I started to walk around the campus and observe the small things. Then I realized that the ramyeon noodle looks similar to the sinusoidal faction so I selected the parent function y=sinx as a starting point.

- To model the repetitive, wavy pattern of the ramen noodles, I selected the most basic wave function, y = sinx, as my parent function.
- The standard sine graph only moves between 1 and -1. However, the actual ripples of the noodle were much taller. To match the thickness and the curves in the photo, I increased the amplitude to 1.77.
- To align the graph with the actual flow of the noodle starting point, I multiplied the x-coefficient by -1. This reflects the graph across the y-axis so that the 'ups and downs' of the wave match the photo perfectly.


Domain and Range:
The domain of my function is {x∈ℝ∣2 < x < 15} because I restricted the model... because I restricted the model to the section of the noodle that is visible in the photograph. Outside of this interval, the graph would represent parts of the noodle that are not shown, so the model would no longer reflect the real situation. Since sine functions are continuous and have no asymptotes, my transformed function is also continuous on this interval. By evaluating the function at the endpoints of the domain, I obtained approximate y-values of −6.47 and −2.93. Therefore, the range of the function is approximately R:{y∈ℝ∣-6.47 < y < -2.93}
